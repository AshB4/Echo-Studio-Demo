PostPunk Pipeline Live Test With Image

This is a short live pipeline test created from PostPunk.

The goal is simple: write one article record, generate the canonical portfolio page, generate RSS, let DEV import it, and then publish the matching DEV draft automatically.

If this appears on DEV, the new PostPunk syndication path is working end to end.

What this test checks

  • The article lives once in PostPunk.
  • The canonical URL points back to the portfolio.
  • The image travels with the article metadata.
  • DEV publishing can be automated after RSS import creates a draft.
